# mini-project-3-API-Powered-App


## What I'm building: (one sentence)
- A country explorer app where users can search and filter countries to view details like population, capital, region, flag, and languages. 

## Which API I'm using: (name and URL)
- REST Countries - restcountries.com 

## Why I chose this: (what interests you about this data?)
- I chose like API because there was a lot of data to work with and it is very visual--every country has a falg, demographic info, and geographic context that makes it interesting for a browsing experience.

## Core features: (3-5 things it will do)
1. Search for any country by name. 
2. Filter countries by regions. (Asia, Europe, Americas, etc...)
3. Display country cards with name, flag, capital, and population.
4. Click a country card to see a detailed view with more info (languages, currency, etc...)
5. Loading and error states handled well

## What I don't know yet: (what about fetch/async/JSON is new to you?)
- Still getting comfortable with how fetch() works with async/await, and how to navigate JSon objects. Like REST countries returns currencies and languages as nested objects so getting all that data and extracting it good might be a problem that I will run into.
